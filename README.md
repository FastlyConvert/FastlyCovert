Fast, simple, privacy-first file conversion & compression — for PDF, images, video, and audio.

## What is FastlyConvert?

FastlyConvert is a web-based toolkit for everyday file tasks:

- **Convert** between common document, image, audio, and video formats
- **Compress** files to reduce size for email, sharing, or storage
- **Create** AI speech outputs (Text-to-Speech) and transcription (Speech-to-Text)

No software installation required — everything runs through the browser.

---

## Featured tools

### 1) Video Compressor
- Input formats: **MP4, MOV, AVI, WebM, MKV**
- Output: **MP4 (H.264)** (best compatibility), optional **WebM** (where available)
- Presets (typical use cases):
  - **Light** — minimal change, safest
  - **Balanced (recommended)** — best size/quality ratio
  - **Strong** — good for strict email/messaging limits
  - **Extreme** — for very small targets (e.g., ~5MB), may show artifacts
- File size limit: **Free up to 500MB** (larger files require Pro)

Try it: https://www.fastlyconvert.com/video-compressor

### 2) AI Text to Speech
- Output format: **MP3**
- Voices: **6 options** (Nova, Echo, Alloy, Onyx, Fable, Shimmer)
- Speed control: **0.25×–4.0×**
- Free trial: **3 runs for new users + 3 runs after signup**
- Privacy: **audio auto-deletes within 24h**, input text **not stored**

Try it: https://www.fastlyconvert.com/text-to-speech

## Privacy & data handling

FastlyConvert is designed to be privacy-first:

- Files are processed only to perform the requested conversion.
- Uploaded files are **automatically deleted within 24 hours**.
- Input text used for TTS/STT is **not stored after processing completes**.
- Transfers are encrypted via **HTTPS/TLS**.
